# 2025fall-graph

## Adjacency Matrix

作为开场题, 定位合理

- 题目描述过于简单直白
- 数据范围 n <= 500 导致 O(n^2) 的输出本身就需要打印 25 万个数字, 对于入门题来说输出量偏大.
- 没有考虑重边, 题目也未明确说明是否存在重边.

## Shortest Path

标准 BFS 求无权图最短路, 选题经典

- 数据范围合理, 但未说明重边和自环.
- 输出格式不友好, 缺乏调试便利.

## Ancestor

DFS 序判断祖先关系, 是树上问题的经典技巧

- 描述含糊, "x is an ancestor of x" 定义突兀.
- 输入格式存在歧义, 边的方向表述不清.
- 样例过于简单, 无法帮助理解复杂情况.
- 最优解需 DFS 序

## Defensive Towers

- 题目背景冗长无实际帮助, 增加阅读负担.
- 核心 insight 隐藏太深, 无提示, 对未学过二分图的学生极不友好.
- 答案不唯一, 但评测方式未明确, 易困惑.

## The Elves

DAG 路径计数+加权求和, 需拓扑排序+DP, 存在严重问题:

- 超纲, 需公式变换和 DP, 对初学者极不友好.
- 题目表述数学化, 理解门槛高.
- 取模运算又是额外负担.

## Sum

状态压缩 DP, 放在 Graph contest 中极不合适:

- 主题不符, 本质是状压 DP, 与图论几乎无关.
- 严重超纲, 难度断层, 初学者无法独立完成.

## 总体评价

1. 难度问题: 前两题过于简单, 后两题过于困难, 缺乏平滑过渡.
2. Sum 题完全偏离图论, The Elves 也更像 DP 题.
3. 缺失经典题型, 无拓扑排序直接应用, 无连通分量, 无环检测等基础题.
4. 题目缺乏提示, 对自学者不友好.
